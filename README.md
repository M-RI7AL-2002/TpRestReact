# Compte React + Spring Boot API

Ce projet est une application simple démontrant comment **Axios** fonctionne avec une **API Spring Boot** pour gérer et afficher une liste de comptes.  
Le frontend en React communique avec le backend via des requêtes REST pour lister, ajouter, mettre à jour et supprimer des comptes.

## Fonctionnalités

- Lister tous les comptes depuis l’API
- Ajouter un compte
- Rafraîchir la liste dynamiquement
- Interface simple et responsive avec Bootstrap

## Backend

- Développé en **Spring Boot**
- Endpoint principal : `http://localhost:8082/comptes`

## Frontend

- Développé en **React**
- Utilise **Axios** pour communiquer avec l’API
- Tableau responsive pour afficher les comptes
